## 개요

직장인을 위한 경력계산 앱<br>
<br>
<br>

## 사용방법

- 연월일에 맞는 근무 시작일을 선택합니다.
- 선택하면 현재 경력과 근무일수, 예상직급을 표시해줍니다.
- 미래 날짜는 선택이 불가합니다.

<br>

## 사용기술

- Dart, Flutter
- CupertinoDatePicker
